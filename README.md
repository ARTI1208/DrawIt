# Draw It!

Simple drawing app written in Flutter. Should work on all platforms, but tested only Android/IOS/Web
 
## Available actions and options:
* Color selection

* History (undo/redo actions)
* Clearing drawing canvas
* Reset offset position

* Shape fill on/off

* Shape fill on/off

## Available tools:
* Move/scale/rotate
* Pencil
* Eraser
* Line
* Rectangle
* Oval
